# Стартовый промт для Claude Code

Скопируй это целиком в Claude Code как первое сообщение:

---

Создай Godot 4 проект с нуля по инструкции в CLAUDE.md.

Начни строго по порядку из раздела "Первый шаг при старте проекта":

1. Создай `project.godot` с настройками для 3D проекта
2. Создай `scenes/game/game_manager.tscn` и `scripts/game/game_manager.gd`
3. Создай карту `scenes/maps/dust2.tscn` — плоская low-poly геометрия с:
   - полом, стенами основных зон (мид, А, Б, коридоры)
   - NavigationRegion3D для ботов
   - коллизиями
4. Создай `scenes/bots/bot.tscn` и `scripts/bots/bot_brain.gd`:
   - FSM с состояниями: IDLE, PATROL, ENGAGE, DEAD
   - бот ходит по случайным точкам (PATROL)
   - видит врага — переходит в ENGAGE, стреляет
5. Камера сверху (60° изометрия), следит за центром карты
6. Базовый HUD: счёт раундов, кнопка паузы

После каждого пункта — сообщи что создал. Не иди дальше пока не скажу "ок".

## Дальнейшие промты (используй по одному когда базовое работает)

**Добавить экономику и фазу покупки:**
> Добавь систему экономики как в CS:GO: деньги за убийства и победу/поражение в раунде. Фаза покупки 15 секунд в начале раунда. Боты автоматически покупают оружие по приоритету. Создай `scripts/game/economy.gd` и подключи к round_manager.

**Добавить гранаты:**
> Добавь три вида гранат: smoke (блокирует видимость ботов через NavObstacle), flash (слепит на 2 сек), frag (урон в радиусе 3м). Игрок может назначить гранаты ботам в паузе. Создай `scripts/weapons/grenade.gd` с базовым классом и тремя наследниками.

**Добавить меню улучшений:**
> Создай экран улучшений `scenes/ui/upgrade_menu.tscn`. После каждого раунда игрок получает очки улучшений. Можно потратить на: aim_level, reaction_time, game_sense, aggression для каждого бота. UI должен показывать всех 5 ботов, их текущие статы и кнопки +.

**Добавить выбор стратегии:**
> В меню паузы добавь выбор командной стратегии: Rush A, Rush B, Split, Default, Eco. Каждая стратегия меняет начальные позиции ботов и их поведение в раунде. Реализуй через `scripts/bots/bot_team.gd`.

**Добавить противников разного уровня:**
> Добавь 4 уровня сложности противников: EASY, MEDIUM, HARD, PRO с разными статами из CLAUDE.md. Перед матчем игрок выбирает уровень. Создай `scripts/game/difficulty.gd`.

**Починить/улучшить AI:**
> Боты [описание проблемы]. Посмотри `scripts/bots/bot_brain.gd` и исправь. Не трогай другие файлы.
