# CS Manager Overview
- Godot 4 tactical top-down CS sandbox with a gameplay-faithful top-down Dust2 rebuild.
- Core loop: buy phase -> live round -> round end with economy and bomb objective.
- Runtime is deterministic by default; RL remains optional and isolated behind `rl_training_mode`.
- Tactical map is code-generated by `TacticalMap` and now owns a tighter Dust2 callout layout, path graphing, authored routes, utility lineups, cover slots, fallback routes, plant slots, retake lanes, bomb cover packages, and sound lanes.
- Match focus is readable bot-vs-bot sandbox behavior around site duties and bomb flow, not manager/meta UI in the current milestone.
- Current milestone adds duty-package planning, bomb-task execution, graph-based movement, closer Dust2 geometry, a drawn icon-first default HUD, and throttled observer-only verbose diagnostics.
